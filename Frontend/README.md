# SAMS Frontend

This is the React frontend for the Smart Attendance Management System.

## 📁 Directory Structure
- `src/components/`: Common UI elements and Layout (Sidebar, Navbar).
- `src/context/`: Auth and Theme context providers.
- `src/pages/`: Main views (Auth, Dashboards, Attendance, Admin, Reports).
- `src/utils/`: API axios instances and helper functions.

## 🛠️ Setup
1. `npm install`
2. `npm start`

## 🎨 Tech Stack
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts
- **Fingerprinting**: FingerprintJS
